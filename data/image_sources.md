## Image Specifications

### Typical Badge Specifications (from research)
- **Format**: PNG with transparent background (preferred), or JPG
- **Minimum Resolution**: 300x300 pixels
- **Recommended Resolution**: 500x500 pixels or higher
- **Physical Badge Sizes**:
  - Peak Award badges: ~65-85mm
  - Milestone badges: ~45mm
  - SIA badges: ~50mm
  - OAS badges: ~50mm
  - Achievement badges: ~40mm
  - Participation badges: ~40mm

## Image Collection Strategy

### Approach for Collecting Badge Images
1. **Manual Collection**: Visit scoutshop.com.au product pages and download product images
2. **Member Access**: Some badges require Scout Member login to view
3. **Alternative Sources**:
   - State/territory Scout websites
   - Official program resource documents
   - Scout group websites (verify authenticity)

### Quality Requirements
- High resolution (minimum 300x300px)
- Clear, well-lit product photography
- Transparent background preferred
- True color representation
- Show complete badge design

## URL Pattern Analysis

### Scout Shop URL Patterns
The Scout Shop uses consistent URL patterns for badge products:

**Pattern**: `https://scoutshop.com.au/products/[badge-name]`

**Naming Convention**:
- Lowercase
- Spaces replaced with hyphens
- Special characters removed or replaced
- Examples:
  - "Animals & Birds" → `animals-and-birds`
  - "Masks & Sculpture" → `masks-sculpture`
  - "Grey Wolf Award" → `grey-wolf-mounting-badge`

### Category URLs
- All Badges: `https://scoutshop.com.au/t/badges`
- Cub Badges: `https://scoutshop.com.au/t/badges/cubs-8-to-10`
- Special Interest Badges: `https://scoutshop.com.au/t/badges/special-interest-badges-multisectional`

## Access Notes

### Restrictions
- Some badge inventory requires Scout Member authentication
- Product availability may vary
- Some badges are discontinued or being phased out (e.g., Sixer badge)

### Alternative Image Sources if ScoutShop Access Limited
1. **Scouts Australia Program Resources**: Official documentation with badge images
2. **State Scout Organizations**: May have badge galleries
3. **Community Resources**: Scout group websites (verify authenticity)
4. **Archived Resources**: Yellow Book website (http://yellowbook.weebly.com/)

## Image Metadata to Collect

For each badge image collected, document:
- Badge ID (matching badges_list.json)
- Image source URL
- Image file path (local storage)
- Image dimensions (width x height)
- File format (PNG/JPG)
- File size
- Image hash (SHA256 for verification)
- Collection date
- Copyright/usage notes
